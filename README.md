# Todo List App para React JS
A continuación de detallarán los pasos para desplegar el proyecto

## Requisitos
- Node js instalado (Mi versión es la 6.14.6).
- Tener disponible el puerto 3000 de tu localhost.

## Despliegue
1. Clonar el proyecto
```sh
git clone https://github.com/Dreamyproud/todo-list-app.git
```
2. Acceder a la consola y ubicarse en la raiz del proyecto clonado. Ejecutar el comando para instalar dependencias.
```sh
npm install
```
3. Una vez instalado todo, ejecutar el compando para iniciar el proyecto.
```sh
npm start
```
4. ¡Disfrutar de nuestro Todo List App!
